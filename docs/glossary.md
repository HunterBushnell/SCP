Glossary

cell
- The NEURON model for a specific cell type (PV, SST, etc.).

tune
- A specific set of parameters and configs for a cell.

trial
- One simulation run within a multi-trial batch.

synapse group
- A named group of synapses with shared config (e.g., pn_exc, bg_inh).

input mode
- The method used to generate spike trains for a synapse group.

snapshot
- A full capture of inputs, traces, and metadata for comparison/debugging.

run_manifest
- The metadata file saved in each output run folder.
